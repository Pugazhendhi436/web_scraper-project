# Python Web Scraper Project

## Features

- Scrapes book information
- Saves data to CSV
- Saves data to SQLite Database
- Sends email alerts for price drops

## Technologies

- Python
- BeautifulSoup
- Requests
- Pandas
- SQLite
- SMTP

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
